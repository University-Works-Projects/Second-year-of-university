1.0 Comandi built-in

    Esterni
        Quando si richiede l'esecuzione di un comando esterno, il corrispondente file eseguibilt
        viene cercato, caricato in memoria ed eseguito.
        Esempio: Il comando ls si trona in /bin/ls

    Interni
        Il omando viene riconosciuto ed eseguito dalla shell.
        Esempi:
            1. echo     # displays all of its arguments to standard output
            1. cd       # change directory
