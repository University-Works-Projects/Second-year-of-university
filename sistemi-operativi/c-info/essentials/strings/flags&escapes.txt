Flags:
    %c      Single character
    %d      Signed decimal integer
    %x      Hexadecimal number
    %f      Decimal floating point number
    %e      Floating point in "scientific notation"
    %s      Character string
    %u      Unsigned decimal integer
    %%      Just print a & sign

Escapes:
    \b      Backspace
    \n      Nweline
    \h      Horizontal tab
    \\      Backslash
    \'      Single quote
    \"      Double quotation
    \xhh    ASCII char specified by digits hh
    \ooo    ASCII cher specified by octal digits ooo
