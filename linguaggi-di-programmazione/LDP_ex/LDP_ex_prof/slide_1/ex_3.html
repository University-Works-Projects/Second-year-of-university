Si consideri il seguente frammento di programma scritto in uno pseudo-linguaggio che usi scoping dinamico e dove la primitiva read(Y) permette di leggere nella variabile Y un intero dall'input standard, mentre write(X) permette di stampare il valore della variabile X. Si dica quali sono i valori stampati.

int X = 0
int Y

void pippo()
    X++
void pluto()
    X++
    pippo()

input(Y)

if Y > 0 then
    int X = 5
    pluto()
else
    pluto()
write(X)


SOLUZIONE:
codice equivalente e commentato:

if case:
    int X = 0
    int Y
    void pippo()
        X++             // 5. X=7 (si fa riferimento alla X di 1.)
    void pluto()
        X++             // 3. X=6 (si fa riferimento alla X di 1.)
        pippo()         // 4.

    input(Y)            // 0. Y=5
    if Y > 0 then
        int X = 5       // 1.
        pluto()         // 2.
    else
        pluto()
    write(X)            // 6. print: 0 (stampa la X globale, quella a 1. Ã¨ morta)

else case:
    int X = 0
    int Y
    void pippo()
        X++             // 4. X=2 (si fa riferimento alla X globale)
    void pluto()
        X++             // 2. X=1 (si fa riferimento alla X globale)
        pippo()         // 3.

    input(Y)            // 0. Y=-5
    if Y > 0 then
        int X = 5
        pluto()
    else
        pluto()         // 1.
    write(X)            // 5. print: 2 (stampa la X globale)
