Good practices

  1. U should declare variable using 'const' (when it's possible) or 'let'. See more here
  2. Charecters '$' and '_' are treats as letters. 2 good practices 4 declaring vars with these simbols are:
    1. Use the '$' simbol as an alias 4 the main functione in a JavaScript library.
    2. Use the '_' simbol as an alias 4 private (hidden) variables
    3. Use methods 2 manipulate strings and not treat them as arrays! (they're not arrays)
    4. When u should use Arrays and when Object?
      • You should use objects when you want the element names to be strings (text).
      • You should use arrays when you want the element names to be numbers.
  3. Since const was introduced, it has become a common practice to declare arrays using const
  4. Do not use for in over an Array if the index order is important. Use instead a for loop, a for of loop, or Array.forEach() when the order is important.
  5. Use "strict mode" by literally typing strict mode at the beginning of a script.

Don't forget!

  1. Type operators: typeof and instanceof
  2. Bitwise Operators
  3. Any variable can be emptied by setting the value to undefined:
    let x = 4; // typeof x return 'number'
    x = undefined; // x's value is now 'undefined' & typeof sx now returns 'undefined'
  4. If you access an object method without (), it will return the function definition. Example here.
  5. Comparing two JavaScript objects always returns false.
  6. let x = Number.MAX_VALUE; // x = Largest possible number in JavaScript let y = Number.MIN_VALUE; // y = Lowest possible number in JavaScript
  7. About arrays and objects:
    • Objects cannot access to array method
    • In JavaScript, arrays only use numbered indexes.
    • In JavaScript, objects use named indexes.
    • JavaScript doesn't support associative arrays. Arrays that try to use named indexes will be converted in objects.
    • In JS, arrays are always an object (so typeof() will return 'objects' both for objects that use named index and arrays that use numbered index).
      To dinstingue an array form an object u need 2 use exampleArr.isArray() method.
  8. JavaScript const variables must be assigned a value when they are declared. Using const without initializing the array will give a syntax error.
  9. Difference between for in and for of:
    /* THE FOLLOWING for CYCLES ARE EQUIVALENT */ const cars = ["BMW", "Volvo", "Mini"];
    let text = "";

    for (let x of cars)
    text += x + "
    ";

    for (let x in cars) text += cars[x] + "
    ";
  10. The continue statement "jumps over" one iteration in the loop.
  11. try, catch, finally, throw statements.
  12. Be careful using the keyword this, pay attenction where u use it! For example it has a different meaning in regular functions and arrow functions.
