  • About CSS

    HTML prevede l'uso di stili CSS in tre modi diversi:

    • Posizionato presso il tag di riferimento
    • Posizionato nel tag <style>
    • Indicato dal tag <link>

    Inoltre HTML permette di assegnare gli stili agli elementi in tre modi:
    • Asseganti a tutti gli elementi di un certo tipo (il nome dell'elemento)
    • Asseganti a tutti gli elementi di una certa categoria (il valore dell'attributo class)
    • Asseganti ad uno specifico elemento (identificato dal valore dell'attributo id)
