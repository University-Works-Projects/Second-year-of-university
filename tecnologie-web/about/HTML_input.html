La maggior parte dei controlli viene definita mediante il tag <input>.
Il suo attributo type stabilisce la tipologia di controllo:

type value What it do Example
type="text" Space 4 enter a text Food:
type="password" Censored text Password:
type="file" Upload a file
type="checkbox" Checkbox
type="radio" Radio button
type="submit" Button to transmit the contents of the form
type="image" Submit button in the form of an image
type="reset" Button to reset all values
type="button" A simple button
type="hidden" hidden field
