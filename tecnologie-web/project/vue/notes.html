Theorical notes

  • A .vue file's structure is:
    
<script>
export default {
    // vue code
}
</script>

<template>
    // html code
</template>

<style>
    // css code
</style>
  • A directive is a special attribute that starts with the v- prefix.
  • A list of som directives:
    1. To bind an attribute to a dynamic value, we use the v-bind directive
    2. The directive v-on is used to listen the DOM events (like a pressed button).
