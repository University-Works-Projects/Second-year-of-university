Theorical notes

  • A .vue file's structure is:
    
<script>
export default {
    // vue code
}
</script>

<template>
    // html code
</template>

<style>
    // css code
</style>
  • A directive is a special attribute that starts with the v- prefix.
  • A list of som directives:
    1. To bind an attribute to a dynamic value, we use the v-bind directive
    2. The directive v-on is used to listen the DOM events (like a pressed button).
  • v-show: To show (or hide) something, you should remember that v-if (temporally) delete the block from the DOM,
    instead the v-show directive, which "simply hide" the current block, instead of deleting it.
    v-if directive is widley used(<-ampiamente utilizzata) for (obvious) efficiency reasons.
  • v-cloak: Use with CSS to hide anything from rendering until the whole Vue application is ready.
